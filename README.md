# 🎬 Movie Rating Website

A modern movie rating website built using React and TMDB API.  
This project displays dynamic movie and TV series data with a responsive and interactive user interface.

---

## 🚀 Overview

Movie Rating Website is a frontend web application that allows users to explore movies and TV series using data from The Movie Database (TMDB) API.

Users can browse trending content, search for movies, view detailed information, and save their favorite titles.

This project was built as a frontend practice project to improve API integration and responsive UI development skills.

---

## ✨ Features

- 🏠 Homepage sections:
  - Movies
  - TV Series
  - Trending
  - Now Playing
- 🔎 Search functionality
- 📄 Detailed movie/series page:
  - Genre
  - Cast
  - Production information
  - Rating
  - Similar movies
- ❤️ Favorite system (stored locally)
- 📱 Fully responsive design
- 🔄 Dynamic data from REST API (TMDB)

---

## 🛠 Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- 🌐 TMDB REST API
- 💻 Pure Frontend (No backend / No database)

## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository-name.git
```

Navigate to the project folder:

```bash
cd your-project-folder
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser at:

```
http://localhost:5173
```

---

## 🔑 Environment Variables

Create a `.env` file and add your TMDB API key:

```
VITE_TMDB_API_KEY=your_api_key_here
```

You can get your API key from:
https://www.themoviedb.org/
